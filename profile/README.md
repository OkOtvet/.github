# OkOtvet.ru 👋

> * Быстрее
> * Дешевле
> * Эффективнее

Оказание услуг для продавцов на меркетплейсах `Ozon`, `Willberies`, `Yandex Market`: 

* автоматизация работы с вопросами и отзывами о товарах без использования API ключей и без платных подписок маркетплейсов
* сбор статистики и аналитики из личного кабинета продавца без использования API ключей
* консультация и разработка веб-расширений по автоматизации работы в личном кабинете продавца
